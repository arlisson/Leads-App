Arquivos de controle

O aplicativo usa arquivos JSON locais para armazenar configurações e manter o comportamento consistente entre execuções.

controle_campos.json

  • Guarda os campos (colunas) por aba.
  • Também guarda qual é a aba selecionada como padrão.

controle_planilha.json

  • Guarda o caminho da última planilha utilizada, para reabrir automaticamente na próxima execução.

controle_ui.json

  • Ícone da janela e ícones dos botões.
  • Tema (cores) e os valores HSL do fundo.
  • Texto do rodapé em HTML (exibido no canto inferior esquerdo).

controle_email_dominios.json

  • Lista de domínios sugeridos para os campos de e-mail.

Recomendação: se for necessário “resetar” configurações, feche o app e apague os arquivos JSON de controle (faça backup antes, se necessário).
