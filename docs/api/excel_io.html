excel_io module¶

excel_io.list_sheets_with_ids(path)[source]¶

Lê um arquivo Excel e retorna uma lista de tuplas contendo o ID da planilha e o nome de cada aba presente no arquivo. O método utiliza a biblioteca openpyxl para carregar o arquivo e iterar sobre as planilhas, extraindo o ID (que pode ser obtido de diferentes atributos dependendo da versão do openpyxl) e o título de cada aba, retornando-os em uma lista estruturada. :param path: Caminho do arquivo Excel a ser lido. :type path: str :returns: Lista de tuplas, onde cada tupla contém o ID da planilha (int) e o nome da aba (str) presente no arquivo Excel.

Return type:

List[Tuple[int, str]]

excel_io.ensure_workbook(path, sheet_name, headers)[source]¶

Garante que o arquivo Excel exista, que a aba exista e que a linha 1 tenha os cabeçalhos (títulos) especificados. :param path: Caminho do arquivo Excel. :type path: str :param sheet_name: Nome da aba onde os dados serão escritos. :type sheet_name: str :param headers: Lista de cabeçalhos (títulos) a serem garantidos na linha 1 da aba. Se a aba já existir, os cabeçalhos serão mesclados com os existentes, mantendo a ordem dos novos e sem duplicatas. :type headers: List[str]

Return type:

None

excel_io.parse_br_number(value)[source]¶

Converte texto numérico para int ou float, aceitando padrão brasileiro.

Exemplos aceitos: - 10 - 10,5 - 1.234,56

Retorna: - int, se for inteiro - float, se tiver parte decimal - None, se não for número válido

excel_io.parse_br_date_or_text(value)[source]¶

Retorna datetime se a data for válida e suportada pelo Excel. Se for anterior a 1900, retorna o texto original. Se não for uma data válida, retorna o texto original.

excel_io.append_row_typed(file_path, sheet_name, campos, row_by_title)[source]¶

Adiciona uma nova linha na planilha, respeitando o tipo de cada campo.

Regras principais: - texto/email/telefone/… -> grava como texto - booleano -> grava como texto - numero:

  • grava como int/float

  • sem forçar casas decimais

  • data:

    • se ano >= 1900, grava como datetime com formato DD/MM/YYYY

    • se ano < 1900, grava como texto

  • campos vazios -> célula vazia

Return type:

None

excel_io.delete_column_by_header(path, sheet_name, header_name)[source]¶

Exclui uma coluna do arquivo Excel com base no nome do cabeçalho. O método verifica se o arquivo e a aba existem, se a linha 1 contém o cabeçalho especificado e, se todas as condições forem atendidas, exclui a coluna correspondente ao cabeçalho fornecido. :param path: Caminho do arquivo Excel. :type path: str :param sheet_name: Nome da aba onde a coluna será excluída. :type sheet_name: str :param header_name: Nome do cabeçalho da coluna a ser excluída. :type header_name: str

Return type:

None

excel_io.rename_column_header(path, sheet_name, old_header, new_header)[source]¶

Renomeia o cabeçalho de uma coluna no arquivo Excel. O método verifica se o arquivo e a aba existem, se a linha 1 contém o cabeçalho antigo especificado e, se todas as condições forem atendidas, atualiza o valor do cabeçalho para o novo nome fornecido. :param path: Caminho do arquivo Excel. :type path: str :param sheet_name: Nome da aba onde a coluna será renomeada. :type sheet_name: str :param old_header: Nome do cabeçalho atual da coluna a ser renomeada. :type old_header: str :param new_header: Novo nome do cabeçalho da coluna. :type new_header: str

Return type:

None
