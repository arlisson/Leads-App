Uso básico

1) Escolher ou criar planilha

  • Clique em Escolher planilha….
  • Selecione um arquivo existente .xlsx ou informe um novo nome para criar.
  • Se escolher um arquivo existente, o app pergunta se você quer importar os campos do cabeçalho (linha 1).

2) Selecionar a aba

  • Use o seletor Aba para escolher em qual aba do Excel será salvo.
  • Use Nova aba para criar uma nova aba no arquivo.
  • Use ↻ Atualizar para recarregar a lista de abas e sincronizar alterações.

3) Preencher e salvar

  • Digite ou cole os dados nos campos.
  • Clique em Salvar (nova linha).
  • Se tudo estiver ok, o app adiciona uma nova linha na planilha e limpa os campos automaticamente.

4) Limpar

Clique em Limpar para apagar os campos preenchidos sem salvar.

5) Gerar Protocolo

  • Clique em Protocolo.
  • Uma tela com o protocolo com o padrão dia mês ano hora minuto segundo será exibido.
  • É possível copiar o protocolo e utilizá-lo como bem entender

Dicas

  • Campos com máscara (telefone/data) foram otimizados para facilitar colar valores.
  • Se você alterar colunas (títulos) no Excel, o app pode atualizar os campos automaticamente quando detectar a mudança.
